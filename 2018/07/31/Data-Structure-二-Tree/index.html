数据结构 - 树

树结构可以帮助我们解决很多重要问题：

  • 代表对象之间有层次依赖的关系
  • 快速查找
  • 提供一种排序的列表数据
  • 文本中的前缀匹配

树种的术语

Root 根节点

树结构中第 0 层的节点称为 root 节点，这个节点作为树的入口

Node 节点

Node 是树结构中的基本单位，是树中分装数据的结构， root 也是一种 node

Leaf 叶子节点

最底层的 node 节点，没有孩子节点
