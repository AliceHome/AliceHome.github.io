链表

链表是数据结构中经常会用到的一种数据结构，是指一串数据项，每一个数据项称之为节点（Node）。

链表主要有两种：

单链表，链表中的每个节点只有向后的索引

双链表，链表除了向后的索引之外，还有指向前一个的索引

定义节点

链表是有节点组成的，首先定义节点的数据结构： Node

  1. 首先将节点定义为一个类 - Node
  2. 添加值，并添加一个初始化方法
  3. 要有一个指针 next 指向下一个节点，注意这个指针必须是可选类型，因为链表最后的一个 Node 的 next 指针不指向任何 Node
